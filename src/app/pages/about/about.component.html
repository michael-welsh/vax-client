About This Website

This website was created as a community service to Hong Kong by Cloudaxis Limited.

Origins

When searching for data across the suite of government websites, we noticed that there were no graphs provided that showed the weekly progress of the vaccination programme, nor any indication of when Hong Kong might reach herd immunity. We decided to take it on as a quick excercise in data scraping and a bit of mathematical calculation.

Sources

Data for the daily doses administered was scraped from the News.gov.hk website in the Covid-19 section.

Other data such as the gender breakdown comees from another Covid-19 website created by the HK government.

We have found that there are discrepancies in the numbers of the two websites, mostly concerning the percentage of the population that has received first and second shots. Those percentages are of course based on the number of inhabitants in Hong Kong, which we counted as 7,496,988. Their estimations of the total doses administered is generally about 2.5% above our estimations.

Tech Stack

For those that care, the frontend of this site was built with Angular 10, the backend with Python 3.8, and it is all running on Google Cloud using the Cloud Run service, which is a brilliant way to quickly run services such as these.
